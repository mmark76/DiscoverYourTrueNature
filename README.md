# Discover Your True Nature

> Working title for an AI-guided self-discovery experience.

The application will guide each user through an adaptive conversation and identify the animal archetype—or combination of archetypes—that best reflects their personality, instincts, strengths, and behaviour under pressure.

## Working tagline

**Discover Your Animal Archetype.**

Greek: **Ανακάλυψε το ζωικό σου αρχέτυπο.**

## Project status

Early discovery and architecture phase. The product name, technology stack, scoring model, and final archetype system remain open to validation.

## Repository structure

- `docs/` — requirements, architecture, decisions, development, and testing documentation.
- `src/app/` — application composition and entry points.
- `src/features/` — product capabilities organised by feature.
- `src/shared/` — reusable code used by multiple features.
- `src/infrastructure/` — external services and technical integrations.

## Development principle

Build the smallest coherent version first, using small modules, clear responsibilities, documented decisions, and controlled incremental changes.
