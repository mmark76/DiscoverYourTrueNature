# Twelve-Archetype Assessment Model

Model version: `12-archetype-v1`

## Purpose and limitations

Animals Within is an entertainment-only self-discovery framework. Its questions, dimensions,
vectors, descriptions, and match scores are editorial and provisional. They are not scientific,
validated, diagnostic, medical, or psychological measurements. A match percentage describes
alignment inside this model only and is not a probability or a share of a total.

## Dimensions

The model uses eight equally weighted dimensions. Each is represented on an editorial `-2` to `+2`
scale:

| ID | -2 endpoint | +2 endpoint |
| --- | --- | --- |
| `affiliation` | independent | community-oriented |
| `reasoning` | intuitive | analytical |
| `tempo` | deliberate | rapid-acting |
| `structure` | adaptive and spontaneous | structured and planned |
| `influence` | supportive and facilitative | directive and leading |
| `exploration` | security and familiarity | novelty and exploration |
| `expression` | reserved | expressive |
| `perspective` | practical and detail-focused | visionary and system-focused |

## Question structure

The assessment contains 24 fixed situational questions: exactly three per dimension. Every question
measures one dimension and presents four answers with values `-2`, `-1`, `+1`, and `+2` once each.
Displayed order varies. IDs and values are language-neutral; English and Greek copy is stored in the
typed localization modules. A run records one answer per question.

## Scoring

1. Add each answer value only to its question's dimension. With three questions, a raw dimension
   total is between `-6` and `+6`.
2. Normalize a user dimension as `rawTotal / 6`.
3. Normalize an archetype dimension as `vectorValue / 2`.
4. Calculate equal-weight root-mean-square distance across the eight normalized dimensions:

   `distance = sqrt(sum((userDimension - archetypeDimension)^2) / 8)`

5. Calculate exact match strength:

   `exactScore = clamp(1 - distance / 2, 0, 1) * 100`

6. Rank all twelve by descending unrounded exact score. Canonical archetype order is the final
   deterministic tie-breaker. Round only the displayed score to a whole number. Scores are not
   normalized to total 100.

The result contains the normalized user dimensions, primary match, secondary match, and one ordered
entry for every canonical archetype.

## Fixtures and balance analysis

The versioned fixture `tests/fixtures/scoring-12-archetype-v1.json` covers canonical profiles,
balanced input, an exact tie, all-negative and all-positive dimensions, and deterministic secondary
profiles. Canonical answers are generated by option value rather than visible position:

- vector `-2`: `[-2, -2, -2]`
- vector `-1`: `[-1, -1, -1]`
- vector `0`: `[-2, +1, +1]`
- vector `+1`: `[+1, +1, +1]`
- vector `+2`: `[+2, +2, +2]`

Run `node scripts/analyzeAssessmentBalance.mjs` to evaluate canonical profiles, seeded samples around
each vector, and a seeded broad sample. The report includes primary/secondary counts, average match
scores, ties, and unreachable archetypes. Unequal random frequencies are reported, not interpreted as
statistical validity.

## Editorial versioning

Any future change to questions, option values, dimensions, vectors, normalization, or tie-breaking
must introduce a new model version and a matching fixture. Copy-only refinements that do not change
scoring should retain the model version but remain aligned between English and Greek.
